Tables

General styles

  • Text should be left aligned. Text td should have class "text"
  • Dates should be right aligned. Date td should have class "date"
  • Numbers should be right aligned. Numbers td should have class "number"
  • Header should be bolded. In case of more headers all headers should be bolded
  • In case of multiple header first header text should be centered and second header should be align based on data type
  • Table should have following classe "report-table table table-striped table-hover table-bordered"
  • In all grids with focus filter, channel that is currently focused should be first (or leftmost) in the channel list

Table sizes

We can append special classes to table that dictates max size for a table
  • "xs-table" >> max width 500px
  • "small-table" >> max width 700px
  • "medium-table" >> max width 900px
  • "large-table" >> max width 1100px
  • "xxl-table" >> max width 1300px
  • If there is no class appended table will take all width of parent element
