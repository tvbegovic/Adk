Messages

  • Unexpected error

    Unexpected error occurred while processing your request. Please try again or contact administrator.

    Example:

            
          <report-message template="Error"></report-message>
  • Work in progress.

    Work in progress. Not for use or review..

    Example:

            
          <report-message template="WorkInProgress"></report-message>
  • No Media house message

    No Media House. Please select a Holding as 'Mine' with some Media Houses. Go to media house

    Example:

            
          <report-message template="NoMediaHouse"></report-message>
  • No Channels In Group.

    No Channels added to group. Please add channels to group. Go to media house

    Example:

            
          <report-message template="NoChannelsInGroup"></report-message>
  • There is no data to display for report

    No data available. Try to expand your filter range.

    Example:

            
          <report-message template="NoData"></report-message>
  • No Day Parts

    You don't have day parts created. Please create your day parts. Go to media house

    Example:

            
          <report-message template="NoDayParts"></report-message>
  • No Key Accounts.

    You don't have key accounts. Please select some key accounts. Go to media house

    Example:

            
          <report-message template="NoKeyAccounts"></report-message>
