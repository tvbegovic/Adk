Dates

For formatting dates the following custom angular filters are being used:

  • userDate - User date format as selected by the user in Settings or My Account.
    The default format is yyyy-MM-dd (e.g. 2001-11-09).
  • weekday - Displays the a date's name of the day of the week based on their locale settings.



Important:

All dates need to be fomated using the userDate filter unless otherwise instructed.



Example:

        
        
    {{CurrentDate | userDate}} //e.g. 2017-12-22
        
    {{CurrentDate | userDate}} //e.g. 22.12.2017

    {{channelDate.PlayDate | weekday}} //e.g. Thursday

    {{channelDate.PlayDate | weekday}} //e.g. četvrtak
