Formating / Numbers

We are using ValueFormatter service for converting value in right format to display. All value converting should be done in this place.

  • Count: No decimal places. thousend separator based on local (5,000 or 5.000) >> ValueFormatter.toLocalString(num, excludeDecimalPalces)
  • Spent: Same as count there is no need to show money simbol (5,000 or 5.000) >> ValueFormatter.toLocalString(num, excludeDecimalPalces)
  • Duration: hh:mm:ss format (33:20:40) >> ValueFormatter.convertSecondsToHourFormat(seconds)
    • Less then minute: 5, 45
    • Less then 10 minutes: 4:03, 4:45
    • Less then hour 22:56, 22:05
    • Less then 10 hours: 9:53:12, 8:01:08
    • More then 10 hours: 234:25:44, 186:13:00, 186:00:02
  • Time in the day: HH:mm:ss. Based on user settings this should be in 0-23 or AM/PM format.
  • Percentages: One decimal place and with percentage symbol (33.3%) >> ValueFormatter.toPercentageString(d)

Example:

      

          <script>
            //for converting seconds to hour format use ValueFormatter.convertSecondsToHourFormat
            var seconds = 5000;
            var formatedSeconds = ValueFormatter.convertSecondsToHourFormat(seconds);

            //To round number to one decimal place use ValueFormatter.roundWithDecimalPlaces
            var percentage = 55.3343;
            var oneDecimalPlacePercentage = ValueFormatter.toPercentageString(percentage);

            //To local string
            var num = 5000;
            var localNumFormat = ValueFormatter.toLocalString(num, true);

          <script>


Available metods in ValueFormatter service:

  • convertSecondsToHourFormat(seconds) >> convertSecondsToHourFormat(70) -> '00:01:10'
  • roundWithDecimalPlaces(num, decimalPlaces) >> roundWithDecimalPlaces(10.36, 1) -> 10.4
  • toPercentageString(num) >> toPercentageString(10.36) -> '10.4%'
  • toLocalString(num, excludeDecimalPalces) >> roundServerNumberString(1200,6787, true) -> 1.201
  • roundServerNumberString(numString) >> roundServerNumberString('1200,6787') -> '1201'
  • convertPieChartDataToDiscreteBarData(piechartData, {sort:true})
